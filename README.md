# Bookyard

A web app for book lovers to rate books and get recommendations.

## built with
- Python
- Flask
- bootstrap
- SQLite
- Pandas

## Initial Dataset
Book-Crossing Dataset, www2.informatik.uni-freiburg.de/~cziegler/BX/

## Deploy
Open the project through Pycharm.
```
 pip install -r requirements.txt
 flask init-db
 flask run
```
