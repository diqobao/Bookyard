# Bookyard

A web app for book lovers to rate books and get recommendations.

## built with
- Python
- Flask
- bootstrap
- SQLite
- Pandas

## Initial Dataset
Book-Crossing Dataset, www2.informatik.uni-freiburg.de/~cziegler/BX/

## Deploy

```
    source venv/bin/activate
    pip install -r requirements.txt
    flask init-db
    flask run
```
